## 1.フローチャート事例

```plantuml
skinparam monochrome true

title カレーライス工程
start

fork
package "カレー"{
    :具材を切る;
    :具材を炒める;
    fork
      repeat:灰汁取り;
      backward:灰汁確認;
      repeat while(灰汁まだでる？)
    fork again
      repeat:具材煮込む;
      backward:具材確認;
      repeat while(具材まだ固い？)
    end fork
}
fork again
package "白米"{
    :米を研ぐ;
    :米を水に浸す;
    :米を炊く;
    :米を蒸らす;
}
end fork

:お皿に白米とカレー入れる;
stop
```

## ER図事例
```plantuml
!define MAIN_ENTITY #E2EFDA-C6E0B4

!define TRANSACTION_MARK_COLOR FFAA00

    BackgroundColor METAL
    BorderColor Black
    ArrowColor Black
}

    entity "PLCマスター" as mplc <<M,TRANSACTION_MARK_COLOR>> MAIN_ENTITY {

        動作時間
        更新日時
    }

    }
```
